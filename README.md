# Bloomberg Financial Solutions

Bloomberg Financial Solutions encompasses the full suite of Bloomberg financial data, analytics, and technology products designed to support front, middle, and back office workflows across asset management, banking, insurance, and corporate treasury.

## APIs

- **Bloomberg Open API (BLPAPI)** - Core market data and reference data API
- **Bloomberg PORT** - Portfolio risk and analytics
- **Bloomberg AIM** - Asset and investment manager OMS
